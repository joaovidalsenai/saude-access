Perguntas Frequentes

Como buscar hospitais próximos?

Use a barra de busca na página inicial para pesquisar por nome do hospital ou localização. O sistema mostrará os hospitais mais próximos automaticamente.

Como funciona o sistema de avaliações?

As avaliações são baseadas em dados oficiais e feedback dos usuários. Cada hospital recebe uma nota de 1 a 5 estrelas baseada em critérios como atendimento, infraestrutura e tempo de espera.

Posso usar o app offline?

Algumas funcionalidades básicas estão disponíveis offline, mas para buscar hospitais e ver informações atualizadas, é necessária conexão com a internet.

Como alterar entre tema claro e escuro?

Clique no ícone do sol/lua no canto superior da tela para alternar entre os temas claro e escuro.

Os dados dos hospitais são atualizados?

Sim, nossa base de dados é atualizada regularmente com informações oficiais dos órgãos de saúde e feedback dos usuários.
